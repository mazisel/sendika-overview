# Sendika

**Türkçe**  
Sendika üyeleri ve yönetim süreçlerini dijital ortama taşıyan kapsamlı yönetim platformu.

**English**  
A comprehensive management platform that digitizes union members and administrative processes.

## Tech Stack
- Next.js + React
- Node.js

## Özellikler / Features
- Üye kayıt ve yönetim sistemi
- Belge ve dosya yönetimi
- Yönetim paneli ve raporlama
- Güvenli erişim kontrolü
